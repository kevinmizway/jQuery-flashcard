JavaScript

JavaScript is a programming language used to make web pages interactive. It runs on your visitor's computer and doesn't require constant downloads from your website.

Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

Variables

JavaScript variables are containers for storing data values.

Variable naming conventions

Coding conventions are style guidelines for programming. They should be followed to improve code readability and make code maintenance easier. Identifier names should use camelCase. All names should start with a letter. You should always put spaces around operators. Always end a statement with a semicolon.
